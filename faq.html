ABOUT ASSIGNMENT DATA ACCESS FAQ

FAQ

1. Can I pre-train my machine learning model on other datasets, e.g. UK Biobank, HCP?

Yes, but please disclose this in your output results.

2. Can I apply image processing tools such as FreeSurfer to derive measures to use as predictors?

Yes, but please disclose this in your output results.

3. Do I need to use deep learning?

No, it is fine to use traditional machine learning methods.

4. Are multiple neuroimaging timepoints used?

No, only clinical scores are measured more than once. Imaging is only at baseline.
